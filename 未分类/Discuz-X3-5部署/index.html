Discuz!X3.5部署
2021-09-09

Discuz!X3.5部署

  • Discuz!X3.5 版本是Discuz!X系列版本的最新测试版，预计于今年(2021年)第四季度发布。相较于3.4版本，该版本对数据库可用字符进行了扩充，增加了邮箱以及短信验证接口，优化了模板等。相比于3.4版本拥有更高的实用性以及安全性！

环境要求

以下四类软件任选其一相互搭配即可。

  • 操作系统
    • Windows
    • Mac OS
    • Linux
  • 服务器软件
    • Apache
    • Nginx
    • IIS
  • 数据库软件
    • MariaDB
    • Mysql
  • 服务器环境
    • PHP

集成环境推荐

  • 宝塔面板
  • 小皮面板

部署示例

步骤一：下载Discuz!X3.5源码文件

  • 访问Discuz!X每日构建或Discuz!X官方仓库下载Discuz!X3.5源码。

步骤二：部署Discuz!X3.5到服务器

  • 将Discuz!X3.5源码包中的upload文件夹内所有文件解压至服务器软件路径，根据所选择的服务器软件类型而异。
  • 使用浏览器打开网站映射路径，进入Discuz!X3.5安装页面。
